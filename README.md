# 🛒 Sistema de Vendas - JavaFX

Este projeto é uma aplicação desktop de *sistema de vendas* desenvolvida com *Java + JavaFX*, como projeto final do curso de Java no SENAI.

A aplicação simula um ambiente completo de gerenciamento comercial, permitindo o controle de clientes, vendedores, produtos, categorias e vendas.

## 🚀 Tecnologias utilizadas

- Java
- JavaFX
- Maven
- SQLite
- Scene Builder
- Arquitetura MVC

### 🚀 Funcionalidades

A aplicação foi desenvolvida como um sistema completo de vendas, contendo:

- 👤 Cadastro e gerenciamento de clientes  
- 🧑‍💼 Cadastro de vendedores  
- 📦 Controle de produtos  
- 🗂️ Gerenciamento de categorias  
- 💰 Registro e controle de vendas  
- 📊 Interface gráfica interativa com JavaFX

### ▶️ Como executar o projeto

#### Pré-requisitos:
- Java JDK 8 ou superior  
- Maven configurado  

#### Passo a passo:

1. Clone o repositório:
git clone https://github.com/seu-usuario/seu-repositorio.git

2. Acesse a pasta do projeto:
cd seu-repositorio

3. Navegue até a estrutura principal:
src/main/java/com/vendas/vendas/

4. Dentro dessa estrutura, localize a pasta:
app/

5. Execute a classe principal:
App.java
